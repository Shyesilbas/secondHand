# Agents

Bu klasor, proje icin calisan AI ajanlarinin davranis ve kapsam kurallarini tutar.

Ilke:
- Kisa oku, kisa karar ver, kisa degistir.
- Önce kurallar, sonra is listesi, sonra kod.
- En az baglamla en guvenli degisikligi yap.

Onerilen dosyalar:
- `rules.md`: Degismeyecek proje kurallari
- `skills.md`: Ajanin kullanacagi yetenek profilleri
- `projects.md`: Modul bazli calisma alanlari
- `runbook.md`: Uygulama sirası ve karar ağacı
